摘要

总结一下自己的Window10安装tensorflow过程。包括jupyter-notebook最简单用法和一个简单tensorflow例子

一、安装Python

先上https://pypi.org/project/tensorflow/#files检查一下支持的Python版本。到目前位置是支持到了3.6，所以从[Python官网](https://www.python.org/downloads/)下载了Python 3.6.5版本 安装Python时，手工选择一下安装目录，如C:\Python36。之后把该路径加到环境变量Path中

以管理员身份打开cmd，进入到C:\Python36，先升级python.exe -m pip install –upgrade pip

二、安装jupyter-notebook

以管理员身份打开cmd，进入到C:\Python36\Scripts，执行pip install jupyter安装

然后继续在C:\Python36\Scripts，执行jupyter notebook启动Jupyter Notebook

执行初始化配置jupyter notebook –generate-config，按提示会生成一个默认配置文件在C:\Users\ZGQ.jupyter目录下

修改jupyter_notebook_config.py的工作目录c.NotebookApp.notebook_dir = ’D:\jupyter’，然后再次启动Jupyter Notebook，这时的根路径已经是刚刚修改的工作路径了。

顺便再安装一个matplotlib，执行pip install matplotlibr安装，然后在Jupyter Notebook上写一个python测试一下

三、安装tensorflow

执行pip install tensorflow安装

执行一个TensorFlow最简单的矩阵乘法

四、Anaconda

安装tensorflow和jupyter-notebook可以通过Anaconda来界面化安装。基于Python 3.6.5的是Anaconda3-5.1.0-Windows-x86_64.exe，下载后下一步下一步就可以装上了，建议一开始选择单用户，安装目录为C:\Anaconda3，之后通过Anaconda安装的执行程序都位于其Scripts目录下。最后一步选择自动配置环境变量，这里不再详细描述。
